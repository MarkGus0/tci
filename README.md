# TCI项目

一个响应式的个人简历/作品集网站，具有iOS设备兼容性优化。

## 特性

- 响应式设计，适配各种设备
- iOS设备兼容性优化
- 现代化的UI/UX设计
- 性能优化
- 跨浏览器支持

## 技术栈

- HTML5
- CSS3
- JavaScript
- jQuery
- 现代化前端框架

## 目录结构

```
├── assets/
│   ├── css/
│   ├── fonts/
│   ├── img/
│   ├── scripts/
│   └── vendors/
└── index.html
```

## 安装和使用

1. 克隆仓库
```bash
git clone https://github.com/MarkGus0/tci.git
```

2. 使用本地服务器运行项目
```bash
python -m http.server 8000
```

3. 在浏览器中访问 `http://localhost:8000`
